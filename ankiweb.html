This description only refers to the version for Anki 2.1. Don't download for the old Anki version 2.0. This add-on offers: - in the Fields Dialog where you manage the fields for a note type this add-on adds Up and Down buttons: You no longer have to enter the new index of a field, you just click up/down. - batch change fonts for ... - ... all fields for current note (from fields window) - ... all fields for all notes (from main window) - ... some fields in the table in the Browser (from main window). You usually set this per note with the Browser Appearance Use at your own risk. So far it works for me. compatibility to other add-ons or newer Anki versions This add-on overwrites "FieldDialog.__init__". It's not compatible with any other add-on that overwrites the same function. So in 2019-11 it's not compatible with Arthur Milchior's 3 add-ons merged: quicker anki, explain deletion, explain database check and Adding note and changing note type become quicker. If Anki ever changes "FieldDialog.__init__" this add-on might cause weird issues. In 2019-11 this function was last changed in 2012. alternatives Have a look at glutanimate's Customize Editor Stylesheet which offers more flexible styling options for the editor. bugs, etc. Feedback, improvements, and error reports are welcome.
